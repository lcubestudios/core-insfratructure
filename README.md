# Linode infrastructure - Terraform

## Download & install
https://www.terraform.io/downloads

## Learn Terraform
https://learn.hashicorp.com/terraform

## Project setup
```
terraform init
```

### Format the configuration
```
terraform fmt
```

### Validate the configuration
```
terraform validate
```
## Create infrastructure

```
terraform apply
```
## Destroy infrastructure
```
Destroy Infrastructure
```
